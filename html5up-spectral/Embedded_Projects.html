  • RC car control using esp8266 wifi - Project Video

×

  • Objective:Developed a WiFi-controlled RC car using the ESP8266 module to enable remote operation via a smartphone

  • Technology:ESP8266 (NodeMCU), L298N motor driver, Arduino IDE, Android app, HTML/JavaScript (for web interface), Li-ion battery.

  • Implementation:Hardware: Modified an RC car by integrating an ESP8266 NodeMCU and L298N motor driver to control DC motors for movement and steering. Powered by a LiPo battery. Software: Programmed ESP8266 using Arduino IDE to create a WiFi access point (AP) or connect to an existing network (STA). Implemented HTTP server for command processing (e.g., forward, backward, left, right). Control: Developed a web-based interface or used an Android app to send HTTP commands to the ESP8266, controlling motor speed via PWM signals.

  • Applications:Enables smartphone-based control of the RC car over WiFi, offering customizable commands, speed control, and potential for additional features like real-time telemetry or camera integration.

Skills: Python, Computer Vision, Object Detection, Real-Time Processing
Libraries Used: OpenCV, YOLO, NumPy, Matplotlib

  • Voice Controlled Car - Project Video

×

  • Objective:Designed a voice-controlled car to enable hands-free operation using voice commands, enhancing accessibility and interactivity for remote control applications.

  • Technology:ESP8266 (NodeMCU), L298N motor driver, Arduino IDE, Android app with speech recognition (e.g., Tasker or custom), HTML/JavaScript (web interface fallback), Li-ion battery.

  • Implementation: Hardware: Integrated ESP8266 NodeMCU and L298N motor driver into an RC car chassis to control DC motors for movement. Powered by a LiPo battery. Software: Programmed ESP8266 using Arduino IDE to establish a WiFi access point or connect to a network. Implemented an HTTP server to process voice-converted commands. Voice Control: Utilized an Android app with speech recognition to convert voice commands (e.g., "go forward," "turn left") into HTTP requests sent to the ESP8266. Added a web interface as a fallback.

  • Applications:Allows hands-free car control via voice commands, improving user experience for hobbyists, accessibility applications, or IoT projects.

Skills: Python, Computer Vision, Object Detection, Real-Time Processing
Libraries Used: OpenCV, YOLO, NumPy, Matplotlib

  • AI Talking Robo - Project Video

×

  • Objective:Developed an AI-powered talking robot to enable interactive voice-based communication in real-time responses.

  • Technology:Arduino uno, L298N motor driver, Arduino IDE, Android app with speech recognition (e.g., Google Speech-to-Text), Python, NLP libraries (e.g., NLTK), Li-ion battery;

  • Implementation:The robot is built using an Arduino Uno to control MG996R servo motors for head rotation and hand shaking. Serial communication is established between the Arduino and an AI module (running on a PC or microcontroller with AI capability) to process user speech and generate responses.

  • Applications:This AI robot can be used for educational demonstrations, STEM learning, reception bots, and interactive exhibitions. It helps students and audiences understand AI-human interaction, robotics control, and embedded system integration in a fun and practical way.

Skills: Python, arduino, Real-Time Processing
Libraries Used: pyaudio, pyttsx, speechify, Serial, vosk, genai, pygame
